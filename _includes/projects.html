Projects

Here is our list of projects:.

  • Neural Voice Cloning

    Project Leads : Mehul Rastogi and Sharad Chitlangia

    This project aims to implement Baidu's paper of the same name which aims to use deep learning models to synthesize a person's voice after only a few samples have been provided

    Check it out

  • Conversion of Neural Networks

    Project Lead : Ajay Subramanian

    This project aimes to build a Python library for the conversion of deep neural networks trained using Tensorflow to GeNN based Spiking neural network architechtures. It uses state-of-the-art algorithms to ensure least performance loss during conversion.

  • Muse GAN

    Project Lead : Het Shah

    This project is based on using Generative networks to combine music from different instruments and then combining them together to make one music piece.

  • Network X

    Project Lead : Rajaswa Patil

    This project aims to build a customizable Targeted Aspect Based Sentiment Analysis for code-mixed social media texts. Advances in this project will help monitoring and analyzing activities on social media platforms in better ways.

  • Financial Markov Decision Project

    Project Lead : Sauradeep Chakraborty

    This project in collaboration with ISI Kolkata aims to apply deep Reinforcement learning to financial trading markets in order to develop a Markov decision process which will suitably capture any general financial market.

  • Analysis and Reconstruction of Music

    Project Lead : Alish Dipani

    This project aims to study and analyze how our brain perceives music and the extraction of meaningful information from the brain signals(EEG). Reconstruction of music using these EEG signals will be done using Machine Learning models.

  • Face Anti-Spoofing

    Project Lead : Pranav Mahajan

    This project aims to use deep learning to prevent false facial verification. It uses auxillary supervision like depth maps and rppg signals so that the model doesn't fit to unfaithful cues and is more interpretable.

  • Spike Time Dependent Plasticity

    Project Lead : Aditya Ahuja

    The main aim of this project is to implement STDP in Spiking Neural Networks. STDP modifies synaptic strength as a function of the relative timing of pre and postsynaptic spikes which enables a neuron to learn synaptic weights.
